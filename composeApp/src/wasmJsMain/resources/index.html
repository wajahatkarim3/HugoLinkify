Medium like Link Previews in Hugo Sites

Link Cards lets you add link previews like Medium.com in your Hugo generated static websites.

Here's how it works:

  • 1. Copy this shortcode into your website's shortcodes.
  • 2. Use this website to generate preview link card of any given URL.
  • 3. Copy the new code of the given URL into your blog posts.
  • 4. You're done!
